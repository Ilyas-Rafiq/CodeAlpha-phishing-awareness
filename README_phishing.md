# Phishing Awareness Training Module

An interactive, browser-based cybersecurity training module that teaches users 
how to recognize, avoid, and report phishing attacks.

Built as part of the **CodeAlpha Cybersecurity Internship**.

---

## Live Demo

Open `phishing_awareness_training.html` directly in any browser — no server needed.

---

## Features

- **Recognize** — Learn the red flags of phishing emails (spoofed senders, 
  urgent language, suspicious links)
- **Real Email Simulations** — Annotated phishing email examples with 
  highlighted suspicious elements and hover tooltips
- **Social Engineering Tactics** — Expandable cards covering Urgency, 
  Authority, Pretexting, Vishing, Smishing, and Spear Phishing
- **Interactive Quiz** — 6 scenario-based questions with instant feedback 
  and a final score
- **Protection Checklist** — Actionable steps to stay safe

---

## What I Learned

- How phishing attacks are structured and what makes them convincing
- Common social engineering tactics used by attackers
- How to identify suspicious URLs, sender addresses, and attachments
- Best practices: MFA, reporting, URL verification, HTTPS misconceptions
